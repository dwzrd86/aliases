## System and Navigation ğŸ—ºï¸

- `l`: Short for `ls`.
- `ls`: Lists files with human-readable sizes and colors.
- `ll`: A more detailed list format.
- `la`: Lists all files, including hidden ones.
- `..`: Navigates to the parent directory.
- `...`: Navigates two directories up.
- `grep`: Highlights matched patterns.
- `df`: Displays disk space usage in a human-readable format.
- `du`: Shows directory space usage in a human-readable format.
- `free`: Displays memory usage in a human-readable format.

## Package Management ğŸ“¦

- `update`: Updates package lists and upgrades distributable packages.
- `upgrade`: Upgrades all packages.
- `install`: Installs a package.
- `dpinstall`: Installs a .deb package.
- `remove`: Removes a package.
- `purge`: Purges a package (removes configuration files).
- `autoremove`: Removes unused packages.
- `search`: Searches for a package.
- `aptlist`: Lists all installed packages.

- `gs`: Shows the git status in a short format.
- `ga`: Adds files to the staging area.
- `gc`: Commits with a message.
- `gp`: Pushes changes to the remote repository.
- `gl`: Displays a summarized log with a graph.
- `gd`: Shows the diff of the current changes.
- `gb`: Lists all branches.
